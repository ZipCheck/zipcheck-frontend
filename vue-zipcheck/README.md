# ZipCheck 메인 히어로 배경 디자인 적용 요청서

## 1. 요청 목적

현재 ZipCheck 홈페이지의 **상단 히어로 영역(검색창이 있는 영역)** 배경이 단색/미세 패턴 수준으로 구성되어 있습니다.

이 영역에 **부동산·지도·감정 이모티콘의 의미가 직관적으로 느껴지는 배경**을 실제로 적용해 주세요.

> 이번 요청은 아이디어 제안이 아니라 **실제 UI 변경이 화면에 보이도록 적용하는 것**이 목적입니다.

---

## 2. 수정 대상 명확화

* 페이지: 홈페이지 ("홈")
* 영역: 상단 히어로 섹션

  * 네비게이션 바 아래
  * 메인 카피("어떤 집을 찾고 계신가요?") + 검색창 전체 영역
* 변경하지 말 것:

  * 텍스트 문구
  * 검색창 위치 및 크기
  * 전체 레이아웃 구조

---

## 3. 배경 디자인 필수 요구사항

### 3.1 부동산 느낌 요소 (필수)

다음 중 **하나 이상 반드시 포함**:

* 아파트/주택을 연상시키는 **단순 실루엣 형태**
* 건물 블록을 추상화한 사각형 패턴
* 실제 사진 ❌, SVG 또는 일러스트 형태 ⭕

배치는:

* 화면 하단 또는 좌/우 가장자리
* 텍스트와 검색창을 방해하지 않는 위치

---

### 3.2 지도 느낌 요소 (필수)

* 실제 지도 이미지 사용 ❌
* 지도에서 연상되는 추상 요소 사용:

  * 도로처럼 보이는 얇은 곡선/직선 라인
  * 블록을 연결하는 네트워크 라인
* 색상: 연한 블루/그레이 계열
* opacity: **3~6%**
* 히어로 영역 전체에 은은하게 적용

---

### 3.3 감정 이모티콘 요소 (선택이지만 권장)

* 서비스의 "지도 감정 표현" 기능을 암시
* 🙂 😐 😡 와 같은 **단순한 이모티콘 아이콘**
* 사용 방식:

  * 매우 작은 크기
  * 반복 패턴 또는 랜덤 배치
  * opacity **2~4%** 수준

---

## 4. 색감 및 분위기 가이드

* 기본 배경: 밝은 블루/연한 그레이 그라데이션
* 예시:

  * 상단: `#F8FAFF`
  * 하단: `#EEF3FF`
* 전체 인상:

  * 신뢰감 있는 부동산 서비스
  * 깔끔하고 차분함
  * 정보 탐색을 방해하지 않음

---

## 5. 구현 방식 지침 (중요)

아래 방식 중 **하나 이상 실제로 사용하여 구현**:

* CSS `background-image` 다중 레이어
* SVG 패턴을 배경 이미지로 적용
* pseudo-element(`::before`, `::after`)를 활용한 오버레이

### ❌ 금지 사항

* 디자인 아이디어 설명만 제공
* 실제 화면에 변화가 없는 상태
* opacity가 너무 낮아 육안으로 구분되지 않는 구현

---

## 6. 완료 판단 기준 (반드시 충족)

다음 질문에 모두 **YES**여야 완료입니다:

* 히어로 영역 배경이 이전과 확실히 달라 보이는가?
* 부동산/지도/감정 표현 서비스라는 인상이 느껴지는가?
* 텍스트와 검색창 가독성은 유지되는가?

---

## 7. 최종 요청 요약

> 상단 히어로 영역 배경에
> **부동산 + 지도 + 감정 이모티콘의 의미가 은은하게 드러나는 배경을 실제로 적용**해 주세요.
