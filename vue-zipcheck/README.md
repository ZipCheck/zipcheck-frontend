# ZipCheck 메인 히어로 배경 미세 보완 요청서

## 옵션 1: 하단 집 프레임만 살짝 강조

---

## 1. 요청 목적

현재 메인 히어로 배경은 전반적으로 완성도가 높고 안정적입니다.
이번 요청은 **디자인 컨셉을 전혀 변경하지 않고**,
화면에 **깊이감과 안정감을 아주 미세하게 추가**하기 위한 보완 작업입니다.

> 목표: 사용자는 거의 인식하지 못하지만,
> 전체 화면이 더 단단해 보이도록 만드는 ‘마지막 조정’

---

## 2. 수정 대상 (고정)

* 페이지: 홈페이지 (Home)
* 영역: 상단 히어로 섹션 배경
* 유지 사항:

  * 집 프레임(Line Art) 디자인
  * 메인 카피 및 검색창 위치/디자인
  * 전체 레이아웃 구조

---

## 3. 핵심 수정 요구사항 (필수)

### 3.1 하단 집 프레임 강조

히어로 영역 **하단 20~30% 영역**에 위치한 집 프레임만 다음과 같이 조정해 주세요.

* opacity: 기존 대비 **+2% ~ +3%**
* 색상 변화 ❌ (기존 색상 유지)
* 선 두께 변화 ❌ (기존 굵기 유지)

> 상단 집 프레임은 **현재 상태 그대로 유지**할 것

---

## 4. 의도 및 기대 효과

* 화면 하단에 시각적 무게 중심이 생김
* 메인 카피 영역은 더 깔끔해 보임
* 배경이 평면처럼 느껴지지 않고, **공간감(depth)** 이 생김

---

## 5. 구현 가이드

* CSS 기준 예시 접근:

  * 하단 영역 집 프레임 레이어에만 opacity 조정
  * 그라데이션 마스크 또는 영역 분리 방식 사용 가능

### ❌ 금지 사항

* 집 프레임 전체 opacity를 동시에 올리는 방식
* 눈에 띄게 강조되는 수준의 변화
* 새로운 배경 요소 추가

---

## 6. 완료 판단 기준

다음 질문에 모두 **YES**여야 합니다.

* 상단은 지금과 거의 동일하게 보이는가?
* 하단 배경이 조금 더 단단하고 안정적으로 느껴지는가?
* 전체 화면이 과하지 않고 더 정제된 인상을 주는가?

---

## 7. 최종 요청 요약

> 메인 히어로 배경의 **하단 집 프레임만 아주 미세하게 강조**하여,
> 전체 화면에 깊이감과 안정감을 추가해 주세요.
> 변화는 작지만, 분명히 느껴져야 합니다.
